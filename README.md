# Portfolio Site (React + Vite)

## Prerequisites

- Node.js 18+ (includes npm)

## Install

```bash
npm install
```

## Development

```bash
npm run dev
```

Vite will print a local URL (usually http://localhost:5173).

## Production build

```bash
npm run build
```

## Preview production build

```bash
npm run preview
```

## Notes for Windows PowerShell

If you see a script execution policy error when running `npm`, use `npm.cmd`:

```bash
npm.cmd install
npm.cmd run dev
```
